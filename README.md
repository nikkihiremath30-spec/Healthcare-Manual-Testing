# Healthcare-Manual-Testing

This project demonstrates manual testing of a healthcare application (OpenMRS).

## Modules Tested
- Login
- Patient Registration
- Patient Search
- Patient List

## Test Scenarios
- Verify login with valid and invalid credentials
- Verify patient registration functionality
- Verify patient search functionality

## Testing Types
- Functional Testing
- UI Testing
- Regression Testing

## Tools Used
- Manual Testing
- Browser Testing
- GitHub
